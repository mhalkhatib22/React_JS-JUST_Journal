# JUST Journal

## Overview
JUST Journal is an innovative, web-based online scientific publishing platform designed to facilitate the dissemination of high-quality research. The system was created to streamline the research paper submission process, eliminate the need for physical disks or hard copies, and significantly accelerate peer-review and publication timelines. By providing a fully digital, cloud-based ecosystem, JUST Journal removes geographical and financial barriers, making scholarly communication more accessible and transparent.

## 🚀 Core Features
* **Role-Based Access Control:** Secure, specialized workflows and dashboards for Authors, Reviewers, and the Editorial Board.
* **Seamless Manuscript Submission:** An intuitive interface allowing authors to easily upload research files, enter metadata (such as titles, abstracts, and research fields), and manage revisions.
* **Real-Time Tracking & Notifications:** Automated alerts that keep all stakeholders continuously informed about submission statuses, review progress, and final decisions.
* **Integrated Peer-Review Workflow:** A structured evaluation phase where assigned experts can review documents anonymously, provide constructive feedback, and submit publication recommendations (accept, reject, or suggest changes).
* **Open Access Publishing:** A platform that ensures approved, peer-reviewed research becomes permanently available to the global scientific community.

## 👥 User Roles
* **Author (Researcher):** Submits original research, tracks the real-time status of their manuscripts, and uploads revised versions based on reviewer feedback.
* **Reviewer (Peer-Review Expert):** Conducts thorough, impartial assessments of assigned manuscripts based on originality and methodology, while identifying any potential ethical concerns.
* **Editorial Board:** Oversees journal operations, conducts initial manuscript screenings, assigns appropriate peer reviewers based on expertise, and makes the final decisions regarding publication.

## 🛠️ Technology Stack
* **Frontend:** React.js
    * Utilized for building interactive, component-based user interfaces and single-page applications (SPAs). 
* **Backend:** Node.js
    * Provides an efficient, event-driven runtime environment for scalable server-side operations and API handling.
* **Database:** MySQL
    * A reliable relational database management system (RDBMS) used for secure data storage, retrieval, and tracking of users, submissions, and reviews.
* **Development Tools:** Visual Studio Code (VS Code)

## 🏗️ Methodology
This project was developed using the **Scrum Agile framework**. The development cycle was divided into short, time-boxed sprints to enhance collaboration, maintain flexibility, and ensure iterative progress from planning through deployment.
